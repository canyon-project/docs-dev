# Canyon Extension

Canyon Chrome 插件，用于手工测试时上报覆盖率数据。

![chrome-crx](/static/documentation/getting-started/first-coverage/chrome-crx.png)

## 安装

[Chrome Web Store](https://chrome.google.com/webstore/detail/canyon-extension/omnpafdjidgpdmlimbangcjjaaodbeof?hl=zh-CN&utm_source=ext_sidebar)

## 使用

reportID、上报间隔时间和reporter可以在popup中配置。测试人员可以在测试过程中，点击插件图标，手工上报覆盖率数据。上报间隔时间大于0时，插件会自动上报覆盖率数据。
