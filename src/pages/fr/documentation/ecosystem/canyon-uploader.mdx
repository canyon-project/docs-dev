# Canyon Uploader

Canyon上报器，用于将本地覆盖率数据上报到Canyon的后台。

## 使用

```shell
curl -LOs https://github.com/canyon-project/uploader/releases/download/v0.0.1/uploader
chmod +x uploader
./uploader
```

它会扫描当前目录下的.canyon_output文件夹，将其中的数据上报到Canyon的后台。
