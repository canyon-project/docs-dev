# 紹介

## 簡単に言えば

Canyon は JavaScript コードカバレッジソリューションです。簡単な Babel プラグインの設定だけで、カバレッジのアップロードとリアルタイムのレポート生成を実現し、開発者と QA エンジニアがエンドツーエンドテストでコードカバレッジを収集するのが難しい問題を解決します。

## アーキテクチャ図

この図は Canyon のアーキテクチャとそのエコシステムの一部を示しています：

1. babel プラグインは CI/CD プロセスでコードのインスツルメンテーションを完了します。
2. プロジェクトはテスト環境にリリースされ、ユーザーは UI 自動化または手動テストを選択し、インスツルメントされたコードのプローブをトリガーします。
3. 生成されたカバレッジデータは Canyon サーバーにアップロードされます。
4. Canyon サーバーはデータを消費し、gitlab が提供するソースコード情報と結合してカバレッジレポートを生成します。
