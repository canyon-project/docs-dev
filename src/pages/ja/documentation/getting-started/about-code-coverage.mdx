# コードカバレッジについて

## コードカバレッジとは何？

コードカバレッジは、テストスイートがどのコード行を実行したかを示す尺度です。四つの次元があります。

- **行カバレッジ**：各コード行が実行されたか？
- **関数カバレッジ**：各関数が呼び出されたか？
- **分岐カバレッジ**：各分岐が実行されたか？
- **文カバレッジ**：各文が実行されたか？

簡単に言えば、コードカバレッジは実行されたソースコードを可視化する尺度で、ソフトウェア開発者が新しいテストをどこに書くべきかを判断するのに役立ちます。より高いカバレッジを実現するために努力できます。

## Canyon がもたらすもの

コードカバレッジツールは開発者にテストを書くことを促し、カバレッジを高めます。テストを書く過程で、開発者はソースコードの新しいエラーや構文上の問題を見つけることができます。これらの問題はアプリケーションをリリースする前に解決することが非常に重要です。

Canyon はエンドツーエンドテストのカバレッジ収集に注力しています。エンドツーエンドテストのカバレッジは、実際のユーザー操作下でアプリケーション全体のコード品質と機能の完全性を確保するために使用されます。ユーザー行動をシミュレートし、統合問題を特定し、リグレッションエラーを防止し、ユーザー体験の最適化に役立つことで、アプリケーションの機能を検証します。このカバレッジは、チームがコードをリリースする際の自信を高め、アプリケーションの実際の使用における安定性と信頼性を確保します。
